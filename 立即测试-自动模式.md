## 🔊 如果没有声音

### 快速检查

1. **检查控制台**（F12）
   - 看是否有 TTS 错误
   - 看是否有 API 错误

2. **检查浏览器设置**
   - Chrome: 地址栏左侧 → 网站设置 → 声音 → 允许

3. **检查系统音量**
   - 确保没有静音
   - 确保音量足够大

4. **检查 API Key**
   - 点击"设置"
   - TTS 提供商 → OpenAI TTS
   - 确认 API Key 正确

### 临时解决方案

如果 OpenAI TTS 不工作，可以临时使用浏览器 TTS：

1. 点击"设置"
2. TTS 提供商 → 浏览器 TTS
3. 保存设置
4. 测试是否有声音

## ⚠️ 重要提示

### 1. 使用耳机 🎧

**强烈建议使用耳机**，否则：
- ❌ 麦克风会录到扬声器声音
- ❌ 可能形成回声
- ❌ 可能无限循环

**确保环境安静**，否则：

- ❌ 可能误触发
- ❌ 识别准确率下降
