# 🎯 连续对话模式 - 快速开始

## 立即体验

### 1️⃣ 启动应用
```bash
cd english-conversation-trainer
npm start
```

应用将在 http://localhost:3000 打开

### 2️⃣ 启用连续模式

在应用界面中：
1. 找到麦克风按钮下方的 **"🔄 进入连续模式"** 按钮
2. 点击按钮
3. 麦克风变为**绿色**，表示连续模式已启用

### 3️⃣ 开始对话

1. **点击绿色麦克风** → 开始录音
2. **说出您的英语句子**
3. **再次点击麦克风** → 停止录音
4. **等待AI回复** → 考拉会说出回复
5. **自动开始录音** → AI说完后0.8秒，自动开始下一轮
6. **继续对话** → 重复步骤2-5

### 4️⃣ 退出连续模式

点击 **"🔄 退出连续模式"** 按钮

## 🎨 视觉指示

- 🟢 **绿色麦克风** = 连续模式已启用
- 🔴 **红色麦克风（脉动）** = 正在录音
- ⚪ **灰色麦克风** = 正在处理
- 🟢 **绿色光环** = 连续模式待机中

## ⚡ 快速提示

### 最佳实践
- ✅ 在安静环境中使用
- ✅ 清晰地说出每个句子
- ✅ 等待AI说完再开始下一句
- ✅ 使用Chrome或Edge浏览器

### 常见问题

**Q: 为什么没有自动开始录音？**  
A: 确保AI已经说完（状态不显示"考拉正在说话..."）

**Q: 可以中途暂停吗？**  
A: 可以，点击"退出连续模式"按钮即可

**Q: 录音时间有限制吗？**  
A: 没有，您可以随时点击麦克风停止录音

## 🔧 配置 OpenAI（可选）

如果您想使用真实的AI对话（而不是测试模式）：

### 方法1：使用配置工具
1. 打开 `setup-openai.html` 文件
2. 输入您的 OpenAI API Key
3. 点击"保存设置"

### 方法2：在应用中配置
1. 点击右上角的"设置"按钮
2. 选择"OpenAI"作为AI提供商
3. 输入您的API Key
4. 点击"保存"

### ⚠️ 安全提醒
**重要**：如果您之前在对话中暴露了API Key，请：
1. 访问 https://platform.openai.com/api-keys
2. 撤销旧密钥
3. 生成新密钥
4. 使用新密钥配置应用
