## 📝 现在需要做什么？

### 第1步：获取 OpenAI API Key（2分钟）

1. 访问：https://platform.openai.com
2. 注册/登录账号
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 复制生成的 Key（格式：sk-xxx...）
6. 充值至少 $5 到账户

### 第2步：启动应用（1分钟）

```bash
cd english-conversation-trainer
npm start
```

等待浏览器自动打开 http://localhost:3000

### 第3步：配置应用（2分钟）

1. 点击右上角的 "设置" 按钮（齿轮图标 ⚙️）
2. 配置 AI 对话：
   - 选择 AI 服务商：**OpenAI**
   - API Key：**[粘贴您的 Key]**
   - 选择模型：**gpt-4o**
3. 配置语音合成：
   - 语音服务商：**OpenAI TTS (推荐)**
   - OpenAI API Key：**[同一个 Key]**
   - 音色选择：**Nova**（推荐）
   - 语音质量：**HD 高清质量**（推荐）
   - 语速调节：**0.9x**（推荐）
4. 点击 "保存设置"

### 第4步：测试（1分钟）

1. 刷新浏览器（Cmd/Ctrl + R）
2. 点击蓝色麦克风按钮
3. 说一句英语："Hello, how are you?"
4. 等待自动提交（约0.5秒）
5. 听考拉用自然的声音回复！✨

## 🆘 遇到问题？

### 常见问题速查

**Q: 语音还是机器人声音？**
- 检查是否选择了 "OpenAI TTS (推荐)"
- 确认 API Key 已正确输入
- 刷新浏览器

**Q: 报错 "API key is required"？**
- 在 "语音合成设置" 中输入 API Key
- 或确保顶部 AI 对话的 Key 已输入

**Q: 报错 "exceeded your quota"？**
- 访问 https://platform.openai.com/account/billing
- 检查余额并充值

**详细解决方案**: 查看 `OpenAI_TTS故障排除.md`
