# Replicate TTS 快速开始

## 5分钟配置指南

### 第1步: 获取 Replicate API Key (2分钟)

1. 打开浏览器，访问 https://replicate.com
2. 点击右上角 **Sign Up** 注册账号（可以用 GitHub 登录）
3. 登录后，点击右上角头像 → **Account Settings**
4. 在左侧菜单选择 **API Tokens**
5. 点击 **Create Token** 按钮
6. 复制生成的 Token（格式：`r8_xxxxxxxxxxxxx`）

💡 **提示**: 保存好这个 Token，后面会用到！

---

### 第2步: 在应用中配置 (1分钟)

1. 打开英语对话训练应用
2. 点击右上角的 **⚙️ 设置** 按钮
3. 滚动到底部找到 **🔊 语音合成设置**
4. 在 **语音服务商** 下拉菜单中选择 **Replicate (高质量)**
5. 在 **Replicate API Key** 输入框粘贴刚才复制的 Token
6. 在 **语音质量** 选择：
   - **Turbo** - 推荐！快速响应，适合日常练习
   - **HD** - 最高质量，声音更自然
7. 点击 **保存设置**

---

### 第3步: 开始使用 (1分钟)

1. 点击麦克风按钮开始录音
2. 说一句英语，比如 "Hello, how are you?"
3. 停止录音
4. 等待 1-3 秒，AI 会用高质量的声音回复你！

🎉 **完成！** 现在你可以享受更自然的 AI 语音了！

## 常见问题

### Q: 为什么有延迟？
A: Replicate 需要在云端生成高质量音频，通常需要 1-3 秒。这是为了获得更好的语音质量。

### Q: 费用是多少？
A: 
- Turbo 模式: 约 $0.01/次
- HD 模式: 约 $0.02/次
- 每天练习 30 分钟约 $0.20-0.40

### Q: 可以切换回免费的浏览器 TTS 吗？
A: 当然！在设置中将 **语音服务商** 改回 **浏览器自带 (免费)** 即可。

### Q: API Key 安全吗？
A: 是的！你的 API Key 只保存在浏览器本地，不会上传到任何服务器。
