## 🚀 快速开始

### 步骤1: 获取 OpenAI API Key

如果您已经有 OpenAI API Key（用于对话），可以直接使用！

如果没有：
1. 访问 https://platform.openai.com
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 充值最低 $5

### 步骤2: 配置应用

1. **打开设置**
   - 点击右上角"设置"按钮

2. **配置 AI 对话**（如果还没配置）
   - 选择 "OpenAI"
   - 输入 API Key
   - 选择模型（推荐 gpt-4o 或 claude-3-5-sonnet）

3. **配置语音合成**
   - 滚动到 "🔊 语音合成设置"
   - 语音服务商：选择 "OpenAI TTS (推荐)"
   - OpenAI API Key：会自动使用上面的 Key
   - 音色选择：推荐 "Nova"（友好活泼）
   - 语音质量：推荐 "HD 高清质量"
   - 语速调节：推荐 0.9x（稍慢，适合学习）

4. **保存设置**
   - 点击"保存设置"

### 步骤3: 测试

1. 刷新浏览器
2. 点击麦克风说话
3. 听考拉用自然的声音回复！
