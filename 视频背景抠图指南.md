## 💡 背景抠图的方法

### 方法 1: 使用透明背景的视频 ⭐⭐⭐⭐⭐
**最推荐的方法**

#### 什么是透明背景视频？
- 视频格式: **WebM** 或 **MOV (ProRes 4444)**
- 背景: 完全透明（Alpha通道）
- 效果: 只显示考拉，背景完全透明

#### 如何制作？

##### 选项 1: 使用 AI 工具自动抠图
1. **Runway ML** (推荐)

   - 费用: 免费试用，付费 $12/月
   - 步骤:
     1. 上传您的考拉视频
     2. 选择 "Remove Background"
     3. 导出为 WebM 格式

2. **Unscreen**
   - 网址: https://www.unscreen.com
   - 功能: 自动视频背景移除
   - 费用: 免费（有水印），付费 $9/月
   - 步骤:
     1. 上传视频
     2. 自动处理
     3. 下载 WebM 格式

3. **Remove.bg Video**
   - 网址: https://www.remove.bg/upload
   - 功能: 视频背景移除
   - 费用: 按次付费

##### 选项 2: 使用专业软件
1. **Adobe After Effects**
   - 使用 Rotobrush 工具
   - 导出为 MOV (ProRes 4444) 或 WebM

2. **DaVinci Resolve** (免费)
   - 使用 Magic Mask 功能
   - 导出为 WebM

#### 导出设置
```

编码: VP9
Alpha通道: 启用
分辨率: 720p 或 1080p
帧率: 30fps
```

#### 使用透明视频

### 方法 2: CSS 背景移除 ⭐⭐⭐
**简单但效果一般**

如果您的视频背景是纯色（如绿幕、蓝幕），可以用 CSS 移除：

#### 步骤
1. 确保视频背景是纯色（绿色或蓝色最好）
2. 我会修改代码添加 CSS 滤镜
3. 效果: 背景变透明

#### 优点
- 不需要重新制作视频
- 实现简单

#### 缺点
- 只适用于纯色背景
- 边缘可能不够干净
- 考拉身上的相似颜色也会被移除

---

### 方法 3: 使用 PNG 序列帧 ⭐⭐⭐⭐
**质量最高但文件大**

#### 什么是序列帧？
- 将视频拆分成多张 PNG 图片
- 每张图片都有透明背景
- 用 JavaScript 快速播放形成动画

#### 如何制作？
1. 使用 After Effects 或 Photoshop
2. 导出为 PNG 序列（带 Alpha 通道）
3. 每秒 24-30 帧

#### 优点
- 质量最高
- 背景完全透明
- 兼容性好

#### 缺点
- 文件很大（可能 50-100MB）
- 加载慢

### 方案 B: CSS 绿幕抠图（如果背景是纯绿色）

#### 1. 确认背景颜色
您的视频背景是什么颜色？
- 绿色 → 可以用这个方法
- 蓝色 → 可以用这个方法
- 其他颜色 → 需要调整参数

#### 2. 修改代码
我会添加 CSS 滤镜：

```css
video {
  /* 移除绿色背景 */
  filter: 
    /* 增加对比度 */
    contrast(1.2)
    /* 移除绿色 */
    hue-rotate(120deg)
    /* 增加饱和度 */
    saturate(0);
}
```

#### 3. 效果
- ⚠️ 边缘可能不够干净
- ⚠️ 需要调整参数
- ✅ 不需要重新制作视频
