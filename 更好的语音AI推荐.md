- **GPT-4 Turbo**: $0.01/1K tokens (输入) + $0.03/1K tokens (输出)

## 🔧 如何切换模型

### 步骤
1. 点击右上角 "设置" 按钮
2. 在 "选择 AI 服务商" 下拉菜单中选择
3. 输入对应的 API Key
4. 选择具体模型
5. 点击 "保存设置"
6. 刷新页面开始使用

### 注意事项
- API Key 保存在浏览器本地，安全可靠
- 可以随时切换不同的模型
- 建议先用免费额度测试
