## 🚀 快速开始（5分钟）

### 第1步：获取 API Key（2分钟）
1. 访问：https://platform.openai.com
2. 注册/登录
3. 创建 API Key
4. 充值 $5-10

### 第2步：配置应用（2分钟）
1. 打开应用设置（右上角齿轮图标）
2. 选择 "OpenAI" 作为 AI 服务商
3. 输入 API Key
4. 选择 "OpenAI TTS (推荐)" 作为语音服务商
5. 选择音色：Nova（推荐）
6. 选择质量：HD
7. 设置语速：0.9x
8. 保存设置

### 第3步：测试（1分钟）
1. 刷新浏览器
2. 点击麦克风
3. 说："Hello, how are you?"
4. 听到自然流畅的语音 ✅

## 💰 费用说明

### OpenAI TTS 定价
- **HD 质量**: $0.030 / 1000 字符
- **标准质量**: $0.015 / 1000 字符

### 实际使用成本
| 使用场景 | HD 质量 | 标准质量 |
|---------|---------|----------|

## 🆘 常见问题速查

### Q1: 语音还是机器人声音？
**快速检查**:
1. 设置中是否选择了 "OpenAI TTS"？
2. API Key 是否正确输入？
3. 是否刷新了浏览器？

**详细解决**: 查看 `OpenAI_TTS故障排除.md` - 问题1

---

### Q2: 报错 "API key is required"？
**快速检查**:
1. 在 "语音合成设置" 中输入了 API Key 吗？
2. 如果使用 OpenAI 对话，顶部也要输入 Key

**详细解决**: 查看 `OpenAI_TTS故障排除.md` - 问题2

---

### Q3: 报错 "Incorrect API key"？
**快速检查**:
1. Key 格式是否正确（sk-xxx...）？
2. 是否有多余的空格？
3. 是否复制完整？

**详细解决**: 查看 `OpenAI_TTS故障排除.md` - 问题3

---

### Q4: 报错 "exceeded your quota"？
**快速检查**:
1. 访问 https://platform.openai.com/account/billing
2. 检查余额是否为 $0
3. 充值至少 $5

**详细解决**: 查看 `OpenAI_TTS故障排除.md` - 问题4

---

### Q5: 没有声音？
**快速检查**:
1. 系统音量是否打开？
2. 浏览器标签页是否静音？
3. 控制台有错误吗？

**详细解决**: 查看 `OpenAI_TTS故障排除.md` - 问题5

## 🎓 学习建议

### 初学者
```yaml
配置:
  音色: Nova
  质量: HD
  语速: 0.8x - 0.9x

原因:
  - 慢速更容易听清
  - HD 质量发音更标准
  - Nova 声音友好，不会紧张
```

### 中级学习者
```yaml
配置:
  音色: Nova 或 Alloy
  质量: HD
  语速: 0.9x - 1.0x

原因:
  - 接近正常语速
  - 可以尝试不同音色
  - 提高听力适应能力
```

### 高级学习者
```yaml
配置:
  音色: 任意（尝试不同口音）
  质量: HD
  语速: 1.0x - 1.2x

原因:
  - 正常或快速语速
  - 适应不同说话风格
  - 挑战听力极限
```
