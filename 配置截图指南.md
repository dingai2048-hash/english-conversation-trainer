# 📸 OpenAI TTS 配置截图指南

## 🎯 跟着截图一步步配置

---

## 第一步：获取 OpenAI API Key

### 1. 访问 OpenAI 官网
- 打开浏览器，访问：https://platform.openai.com
- 点击右上角 "Sign Up" 或 "Log in"

### 2. 进入 API Keys 页面
- 登录后，点击左侧菜单的 "API keys"
- 或直接访问：https://platform.openai.com/api-keys

### 3. 创建新的 API Key
- 点击 "+ Create new secret key" 按钮
- 给 Key 起个名字，例如："English Trainer"
- 点击 "Create secret key"

### 4. 复制 API Key
- **重要**：立即复制生成的 Key（格式：sk-xxx...）
- 保存到安全的地方（只显示一次！）

### 5. 充值账户
- 点击左侧菜单的 "Billing"
- 点击 "Add payment method" 添加支付方式
- 充值至少 $5（推荐 $10-20）

---

## 第二步：启动应用

### 1. 打开终端
```bash
cd english-conversation-trainer
npm start
```

### 2. 等待启动
- 看到 "Compiled successfully!" 
- 浏览器自动打开 http://localhost:3000

---

## 第三步：配置应用

### 1. 打开设置界面

**在应用中找到：**
- 右上角有一个 "设置" 按钮（齿轮图标 ⚙️）
- 点击它

**您会看到：**
- 一个弹出窗口
- 标题："AI API 设置"
- 副标题："配置你的AI对话服务"

---

### 2. 配置 AI 对话模型

**找到第一个部分：**
```
选择 AI 服务商
[下拉菜单]
```

**操作：**
1. 点击下拉菜单
2. 选择 "OpenAI"

**然后会出现：**
```
API Key
[输入框]
```

**操作：**
1. 点击输入框
2. 粘贴您的 OpenAI API Key（sk-xxx...）

**接着会出现：**
```
选择模型
[下拉菜单]
```

**操作：**
1. 点击下拉菜单
2. 选择 "gpt-4o" 或 "gpt-4-turbo"（推荐 gpt-4o）

---

### 3. 配置语音合成

**向下滚动，找到：**
```
🔊 语音合成设置
选择AI说话的声音来源
```

**第一个选项：**
```
语音服务商
[下拉菜单]
```

**操作：**
1. 点击下拉菜单
2. 选择 "OpenAI TTS (推荐)"

**会出现绿色提示框：**
```
✨ OpenAI TTS：性价比最高的选择！
声音自然流畅，价格便宜...
```

**然后会出现更多选项：**

#### 选项 1: OpenAI API Key
```
OpenAI API Key
[输入框]
输入你的 OpenAI API Key（与对话AI共用）
```

**操作：**
- 粘贴同一个 OpenAI API Key
- 或者留空（会自动使用上面的 Key）

#### 选项 2: 音色选择
```
音色选择
[下拉菜单]
```

**操作：**
1. 点击下拉菜单
2. 看到 6 个选项：
   - Nova - 女声、友好、活泼（推荐）⭐
   - Shimmer - 女声、温柔、甜美
   - Alloy - 中性、清晰、适合教学
   - Echo - 男声、专业、沉稳
   - Fable - 英式口音、优雅
   - Onyx - 男声、深沉、权威
3. 选择 "Nova"（推荐）

#### 选项 3: 语音质量
```
语音质量
[下拉菜单]
```

**操作：**
1. 点击下拉菜单
2. 看到 2 个选项：
   - HD 高清质量 - $0.030/1K字符（推荐）⭐
   - 标准质量 - $0.015/1K字符（更便宜）
3. 选择 "HD 高清质量"（推荐）

#### 选项 4: 语速调节
```
语速调节: 0.9x
[滑块]
慢速 (0.5x)    正常 (1.0x)    快速 (1.5x)
```

**操作：**
1. 拖动滑块到 0.9x
2. 或者直接点击 0.9 的位置

**下方提示：**
```
推荐 0.9x，稍微慢一点更适合学习
```

---

### 4. 保存设置

**滚动到底部，找到：**
```
[取消]  [保存设置]
```

**操作：**
1. 点击蓝色的 "保存设置" 按钮
2. 看到提示："设置已保存！"
3. 设置窗口自动关闭

---

## 第四步：测试

### 1. 刷新页面
- 按 `Cmd + R` (Mac)
- 或 `Ctrl + R` (Windows)

### 2. 开始对话

**您会看到：**
- 中间有一个大大的考拉视频/图片
- 下方有一个蓝色的麦克风按钮
- 文字："Tap to speak"

**操作：**
1. 点击蓝色麦克风按钮
2. 麦克风变成红色，开始录音
3. 说一句英语，例如："Hello, how are you?"
4. 说完后等待 0.5 秒，自动提交

**您会看到：**
1. 文字变成："正在处理..."
2. 您的话出现在右侧对话框（蓝色气泡）
3. 考拉开始"思考"（视频切换）
4. 考拉的回复出现（灰色气泡）
5. **听到自然流畅的语音！**（不是机器人声音）

### 3. 检查效果

**成功的标志：**
- ✅ 语音听起来像真人
- ✅ 发音清晰自然
- ✅ 语速适中（0.9x）
- ✅ 音色友好活泼（Nova）
- ✅ 没有机器人感

**如果还是机器人声音：**
- ❌ 可能还在使用浏览器自带语音
- 🔧 重新检查设置中是否选择了 "OpenAI TTS"
- 🔧 确认 API Key 已正确输入
- 🔧 刷新浏览器重试

## 🆘 遇到问题？

### 问题 1: 找不到设置按钮
**位置：**
- 桌面版：右上角，齿轮图标 ⚙️
- 移动版：底部导航栏，最右边

### 问题 2: 没有 "OpenAI TTS" 选项
**原因：**
- 代码可能没有更新
- 刷新浏览器（Cmd/Ctrl + Shift + R 强制刷新）

### 问题 3: 保存后没反应
**检查：**
1. 打开浏览器控制台（F12）
2. 查看 Console 标签
3. 看是否有红色错误信息
4. 截图发给开发者

### 问题 4: 语音还是机器人
**检查：**
1. 重新打开设置
2. 确认 "语音服务商" 显示 "OpenAI TTS (推荐)"
3. 如果不是，重新选择并保存
4. 刷新浏览器

### 问题 5: 报错 "API key is required"
**解决：**
1. 确认在 "OpenAI API Key" 输入框中输入了 Key
2. 确认 Key 格式正确（sk-xxx...）
3. 保存设置
4. 刷新浏览器
