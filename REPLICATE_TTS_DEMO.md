# Replicate TTS 功能演示

## 🎯 功能概述

Replicate TTS 是一个云端高质量语音合成服务，可以让 AI 考拉用更自然、更真实的声音说话。

## 🆚 效果对比

### 浏览器 TTS（默认）
```
用户: "Hello, how are you?"
AI: "I'm doing great! How about you?" 
    [机械的电脑声音，即时播放]
```

### Replicate TTS（高质量）
```
用户: "Hello, how are you?"
AI: "I'm doing great! How about you?"
    [自然的人声，1-3秒后播放，声音流畅自然]
```

## 🎬 使用流程演示

### 场景1: 首次配置

```
步骤1: 用户点击"设置"按钮
      ↓
步骤2: 滚动到"语音合成设置"
      ↓
步骤3: 选择"Replicate (高质量)"
      ↓
步骤4: 输入 Replicate API Key
      ↓
步骤5: 选择"Turbo"模式
      ↓
步骤6: 点击"保存设置"
      ↓
步骤7: 看到"设置已保存！"提示
```

### 场景2: 日常使用

```
步骤1: 点击麦克风按钮
      ↓
步骤2: 说英语："Hello, how are you?"
      ↓
步骤3: 再次点击麦克风停止录音
      ↓
步骤4: 看到"正在处理..."
      ↓
步骤5: AI文字回复出现："I'm doing great!"
      ↓
步骤6: 等待1-3秒（生成语音）
      ↓
步骤7: 听到自然的AI声音说话
      ↓
步骤8: 继续对话...
```

### 场景3: 切换回免费TTS

```
步骤1: 点击"设置"按钮
      ↓
步骤2: 在"语音服务商"选择"浏览器自带 (免费)"
      ↓
步骤3: 点击"保存设置"
      ↓
步骤4: 现在使用免费的浏览器TTS
```

## 🔧 故障排除演示

### 问题: 语音无法播放

```
检查步骤:
1. 打开浏览器控制台 (F12)
2. 查看是否有错误信息
3. 检查 API Key 是否正确
4. 测试网络连接
5. 尝试切换回浏览器 TTS

解决方案:
- 如果是 API Key 错误 → 重新输入
- 如果是网络问题 → 检查连接
- 如果是服务问题 → 稍后再试
```

### 问题: 延迟太长

```
当前状态: HD 模式，延迟 3-5 秒

优化步骤:
1. 打开设置
2. 将"语音质量"改为"Turbo"
3. 保存设置
4. 重新测试

结果: 延迟降低到 1-2 秒
```
