# 🐨 如何添加你的考拉图片

## 快速步骤

1. **准备图片**
   - 找一张可爱的考拉图片
   - 推荐尺寸：500x500 像素或更大
   - 格式：PNG 或 JPG

2. **重命名**
   ```
   将图片重命名为：koala.png
   ```

3. **放到这个文件夹**
   ```
   就是这个 public 文件夹！
   把 koala.png 放在这里
   ```

4. **刷新浏览器**
   - 按 Cmd+R (Mac) 或 Ctrl+R (Windows)
   - 你的考拉就会出现了！

## 当前文件夹位置
```
english-conversation-trainer/public/
├── koala.png          ← 把你的图片放这里
├── favicon.ico
├── index.html
└── ...
```

## 测试图片
上传后，在浏览器访问：
```
http://localhost:3000/koala.png
```

能看到图片就说明成功了！
