# 🚀 快速测试指南

## 立即开始测试混合模式

### 1️⃣ 启动应用（30秒）

```bash
cd english-conversation-trainer
npm start
```

等待浏览器自动打开 `http://localhost:3000`

### 2️⃣ 配置 Whisper（1分钟）

1. 点击右上角 **⚙️ 设置**
2. 找到 **STT 提供商**，选择 **Whisper**
3. 确认 **OpenAI API Key** 已填写
4. 点击 **保存设置**
5. 关闭设置窗口

### 3️⃣ 开始测试（2分钟）

1. **点击蓝色麦克风按钮** 🎤
2. **说英语**："Hello, how are you today?"
3. **停顿 1.5 秒** ⏱️
4. **观察**：
   - 🎤 正在录音... → 🔄 正在识别... → 🗣️ 考拉正在说话...
5. **AI 回复后自动开始下一轮录音**
6. **继续对话**："I want to practice English."
7. **停顿 1.5 秒**
8. **重复步骤 4-7**
9. **点击红色停止按钮** 🛑 结束对话

### 4️⃣ 验证结果（30秒）

✅ **成功标准**：
- 说完话后 1.5 秒自动提交
- 识别结果准确（95%+）
- AI 正常回复
- 自动开始下一轮录音
- 状态提示清晰

❌ **失败标准**：
- 卡在"正在录音..."不动
- 识别结果错误
- 需要手动点击停止
- 没有自动开始下一轮

## 🐛 遇到问题？

### 问题：卡在"正在录音..."

**解决**：
1. 打开浏览器控制台（F12）
2. 查看错误信息
3. 检查 API Key 是否正确
4. 检查网络连接

### 问题：识别不准确

**解决**：
1. 说话清晰，语速适中
2. 减少背景噪音
3. 使用高质量麦克风

### 问题：静音检测不灵敏

**解决**：
1. 说完话后明显停顿
2. 等待完整的 1.5 秒
3. 不要在停顿时发出声音
