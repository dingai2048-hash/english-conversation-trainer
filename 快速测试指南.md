# 🚀 快速测试指南

## ⚡ 立即开始测试（3 步）

### 第 1 步：刷新浏览器 🔄

**必须刷新才能加载修复后的代码！**

```bash
# Mac 用户
按 Cmd + R

# Windows 用户  
按 Ctrl + R

# 如果还不行，试试硬刷新（清除缓存）
# Mac: Cmd + Shift + R
# Windows: Ctrl + Shift + R
```

### 第 2 步：点击麦克风 🎤

- 蓝色圆形按钮
- 点击后变红色
- 浏览器可能会请求麦克风权限（允许它）

### 第 3 步：说英语 🗣️

**试试这些简单的句子**：

```
✅ "Hello, how are you?"
✅ "My name is [你的名字]"
✅ "I like music"
✅ "What's your name?"
✅ "Nice to meet you"
```

**说完后**：
- 等待 0.5-1 秒
- **不要手动点击麦克风**
- 让系统自动检测句子结束

## ✅ 成功的标志

如果一切正常，您会看到：

1. **麦克风自动变蓝** ✅
2. **显示"正在处理..."** ✅
3. **您的话出现在对话框（蓝色气泡）** ✅
4. **AI 回复出现（灰色气泡）** ✅
5. **听到自然的语音** ✅
6. **考拉视频切换状态** ✅

## 🔄 测试连续对话模式

### 启用连续模式

1. 点击"🔄 进入连续模式"按钮
2. 说第一句话
3. AI 回复后，会自动开始录音
4. 继续对话，无需手动点击

### 退出连续模式

- 点击"🔄 退出连续模式"按钮

## ❌ 如果不工作

### 快速检查清单

- [ ] 已经刷新浏览器了吗？（Cmd + R）
- [ ] 麦克风权限允许了吗？
- [ ] 系统音量打开了吗？
- [ ] 浏览器标签页没有静音吗？
- [ ] 使用的是 Chrome 或 Edge 浏览器吗？

### 查看错误信息

1. **打开开发者工具**
   - Mac: `Cmd + Option + I`
   - Windows: `F12`

2. **切换到 Console 标签**

3. **查看红色错误信息**

4. **复制错误信息告诉我**

### 使用调试工具

```bash
# 在浏览器中打开
open english-conversation-trainer/调试工具.html
```

点击按钮检查：
- ✅ 检查配置

## 🆘 常见问题

### Q1: 说完话没反应？

**解决方案**：
1. 确认已经刷新浏览器
2. 查看控制台是否有错误
3. 使用调试工具检查配置

### Q2: 没有声音？

**解决方案**：
1. 检查系统音量
2. 检查浏览器标签页是否静音
3. 右键点击标签页 → 取消静音

### Q3: API Key 错误？

**解决方案**：
1. 打开 `自动配置OpenAI.html`
2. 点击"自动配置"
3. 刷新应用

### Q4: 麦克风权限被拒绝？

**解决方案**：
1. 点击地址栏左侧的锁图标
2. 找到"麦克风"权限
3. 设置为"允许"
4. 刷新页面
