## 📝 配置步骤（3分钟完成）

### 第1步：获取 OpenAI API Key

1. 访问：https://platform.openai.com
2. 点击右上角 "Sign Up" 注册（如果已有账号直接登录）
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 给 Key 起个名字（例如：English Trainer）
6. 复制生成的 API Key（格式：sk-xxx...）
7. **重要**：充值至少 $5 到账户

### 第2步：在应用中配置

1. **启动应用**
   ```bash
   cd english-conversation-trainer
   npm start
   ```

2. **打开设置**
   - 点击右上角的 "设置" 按钮（齿轮图标⚙️）

3. **配置 AI 对话**（如果还没配置）
   ```
   选择 AI 服务商: OpenAI
   API Key: [粘贴您的 OpenAI API Key]
   选择模型: gpt-4o 或 gpt-4-turbo
   ```

4. **配置语音合成**
   - 滚动到 "🔊 语音合成设置" 部分
   - 选择：
     ```
     语音服务商: OpenAI TTS (推荐)
     OpenAI API Key: [粘贴同一个 API Key]
     音色选择: Nova - 女声、友好、活泼（推荐）
     语音质量: HD 高清质量（推荐）
     语速调节: 0.9x（稍慢，适合学习）
     ```

5. **保存设置**
   - 点击底部的 "保存设置" 按钮
   - 看到 "设置已保存！" 提示

### 第3步：测试

1. **刷新页面**
   - 按 `Cmd + R` (Mac) 或 `Ctrl + R` (Windows)

2. **开始对话**
   - 点击蓝色麦克风按钮
   - 说一句英语："Hello, how are you?"
   - 等待自动提交（约0.5秒）
   - 听考拉用自然的声音回复！

## 🔧 常见问题

### Q1: 可以用同一个 API Key 吗？
**A**: 可以！OpenAI 对话和 TTS 可以使用同一个 API Key。

### Q2: 如何知道配置成功？
**A**: 
1. 保存设置后没有报错
2. 开始对话，AI 能正常回复
3. 语音听起来自然流畅（不是机器人声音）

### Q3: 语音听起来还是机器人？
**A**: 检查：
1. 是否选择了 "OpenAI TTS"（不是 "浏览器自带"）
2. 是否输入了正确的 API Key
3. 是否选择了 HD 质量
4. 刷新浏览器重试

### Q4: 报错 "API key is required"？
**A**: 
1. 检查是否在 "语音合成设置" 中输入了 API Key
2. 如果使用 OpenAI 对话模型，确保在顶部也输入了 API Key
3. 保存设置后刷新页面

### Q5: 报错 "OpenAI TTS API error"？
**A**: 
1. 检查 API Key 是否正确
2. 检查账户是否有余额
3. 检查网络连接
4. 按 F12 查看控制台详细错误
