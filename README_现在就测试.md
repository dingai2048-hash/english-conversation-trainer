## 🔍 如果遇到问题

### 视频不播放？
1. 打开浏览器控制台（F12）
2. 查看 Console 标签是否有错误
3. 查看 Network 标签，确认视频文件已加载
4. 尝试直接访问视频：`http://localhost:3000/koala.mp4`

### 视频切换不流畅？
- 您的视频文件比较大（每个约7.6MB）
- 建议压缩到2-3MB以提升性能
- 查看 `视频功能完成状态.md` 中的压缩方法

### 视频显示为图片或emoji？
- 这是正常的后备机制
- 检查视频文件是否在 `public/` 文件夹中
- 检查文件名是否正确（小写，.mp4后缀）
