# 🚀 配置向导 - 一步步设置您的应用

## 📋 配置清单

我们需要配置两个部分：
1. ✅ AI 对话模型（推荐 Claude 3.5 Sonnet）
2. ✅ 语音合成（OpenAI TTS）

---

## 第一步：获取 API Keys

### 选项 A: 使用 Claude + OpenAI TTS（最推荐）

#### 1. 获取 Claude API Key
1. 访问：https://console.anthropic.com
2. 点击右上角 "Sign Up" 注册账号
3. 验证邮箱
4. 进入 "API Keys" 页面
5. 点击 "Create Key"
6. 复制 API Key（格式：sk-ant-xxx）
7. 充值最低 $5

#### 2. 获取 OpenAI API Key
1. 访问：https://platform.openai.com
2. 注册/登录账号
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 复制 API Key（格式：sk-xxx）
6. 充值最低 $5

### 选项 B: 只使用 OpenAI（简单方案）

#### 获取 OpenAI API Key
1. 访问：https://platform.openai.com
2. 注册/登录账号
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 复制 API Key（格式：sk-xxx）
6. 充值最低 $5

**注意**：同一个 OpenAI Key 可以用于对话和语音！

---

## 第二步：配置应用

### 1. 打开应用
在浏览器中访问：http://localhost:3000

### 2. 打开设置
点击右上角的 "设置" 按钮（齿轮图标）

### 3. 配置 AI 对话

#### 如果使用 Claude（推荐）
```
选择 AI 服务商: Anthropic Claude
API Key: [粘贴您的 Claude API Key]
选择模型: claude-3-5-sonnet-20241022
```

#### 如果使用 OpenAI
```
选择 AI 服务商: OpenAI
API Key: [粘贴您的 OpenAI API Key]
选择模型: gpt-4o
```

### 4. 配置语音合成

滚动到 "🔊 语音合成设置" 部分：

```
语音服务商: OpenAI TTS (推荐)
OpenAI API Key: [粘贴您的 OpenAI API Key]
音色选择: Nova - 女声、友好、活泼（推荐）
语音质量: HD 高清质量 - $0.030/1K字符（推荐）
语速调节: 0.9x（稍慢，适合学习）
```

### 5. 保存设置
点击底部的 "保存设置" 按钮

---

## 第三步：测试

### 1. 刷新页面
按 `Cmd + R` (Mac) 或 `Ctrl + R` (Windows)

### 2. 开始对话
1. 点击蓝色的麦克风按钮
2. 说一句英语，例如："Hello, how are you?"
3. 说完后等待自动提交（约0.5秒）
4. 听考拉用自然的声音回复！

### 3. 检查效果
- ✅ AI 回复是否聪明、自然？
- ✅ 语音是否流畅、像真人？
- ✅ 录音是否自动提交？
- ✅ 考拉说话时是否停止录音？

## 💡 配置技巧

### 1. API Key 安全
- API Key 只保存在浏览器本地
- 不会上传到任何服务器
- 可以随时更换

### 2. 测试不同音色
在设置中尝试不同的音色：
- Nova（友好活泼）
- Shimmer（温柔甜美）
- Alloy（专业清晰）

### 3. 调整语速
- 初学者：0.8x - 0.9x
- 中级：0.9x - 1.0x
- 高级：1.0x - 1.2x

## 🆘 常见问题

### Q1: API Key 在哪里输入？
**A**: 点击右上角"设置"按钮，在弹出的窗口中输入。

### Q2: 需要两个 API Key 吗？
**A**: 
- 如果用 Claude + OpenAI TTS：需要两个
- 如果全用 OpenAI：只需要一个

### Q3: API Key 格式是什么？
**A**: 
- Claude: sk-ant-xxx
- OpenAI: sk-xxx

### Q4: 如何知道配置成功？
**A**: 
1. 保存设置后不报错
2. 开始对话，AI 能正常回复
3. 语音听起来自然流畅

### Q5: 配置后没反应？
**A**: 
1. 检查 API Key 是否正确
2. 检查账户是否有余额
3. 刷新浏览器
4. 按 F12 查看控制台错误
