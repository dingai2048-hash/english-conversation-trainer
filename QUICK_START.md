# 快速开始指南

## 🚀 立即开始使用

### 0. 配置AI API（可选）

**默认使用Mock模式**（测试模式），AI会返回预设的回复。

**如果想要真实的AI对话**：
1. 点击右上角的"设置"按钮 ⚙️
2. 选择AI服务商（推荐OpenAI或豆包）
3. 输入API Key
4. 保存设置

详细配置指南：[API_SETUP_GUIDE.md](./API_SETUP_GUIDE.md)

### 1. 打开应用
应用已经在运行：http://localhost:3000

### 2. 允许麦克风权限
首次使用时，浏览器会询问麦克风权限，请点击"允许"。

### 3. 开始对话

#### 步骤：
1. **点击麦克风按钮** 🎤
   - 按钮会变红，表示正在录音
   - 屏幕显示"正在录音..."

2. **对着麦克风说英语** 🗣️
   - 清晰地说出你想说的话
   - 例如："Hello, how are you today?"

3. **再次点击麦克风停止** ⏹️
   - 系统开始识别你的语音
   - 屏幕显示"正在处理..."

4. **查看你的消息** 💬
   - 你说的话会显示在右侧对话区
   - 蓝色气泡，右对齐

5. **听考拉回复** 🐨🔊
   - 考拉会用英语"说话"回复你
   - 同时显示文字内容
   - 灰色气泡，左对齐

6. **查看翻译（可选）** 🌏
   - 点击右上角的翻译按钮
   - 显示/隐藏中文翻译

## 💡 使用技巧

### 录音技巧
- ✅ 在安静的环境中使用
- ✅ 麦克风距离嘴巴15-30厘米
- ✅ 说话清晰，语速适中
- ✅ 避免背景噪音

### 对话技巧
- 🎯 从简单的问候开始："Hello", "Hi"
- 🎯 使用完整的句子
- 🎯 等考拉说完再开始下一轮
- 🎯 如果识别错误，重新录音即可

### 学习技巧
- 📚 听考拉的发音，模仿学习
- 📚 打开翻译功能理解意思
- 📚 重复练习相同的句型
- 📚 逐渐增加句子复杂度

## 🎨 界面说明

### 左侧区域
- **考拉角色**: 显示当前状态
  - 正常：空闲状态
  - 缩放动画：正在听你说话
  - 脉冲动画：正在思考或说话
- **麦克风按钮**: 控制录音
  - 灰色：空闲，点击开始录音
  - 红色：录音中，点击停止
  - 禁用：处理中或说话中
- **状态文字**: 显示当前操作状态

### 右侧区域
- **对话记录标题栏**: 
  - 左侧：标题
  - 右侧：翻译切换按钮
- **对话显示区**: 
  - 蓝色气泡：你的消息
  - 灰色气泡：AI的消息
  - 自动滚动到最新消息
  - 显示时间戳

## ⚠️ 常见问题

### 问题1：麦克风不工作
**解决方案**:
1. 检查浏览器权限设置
2. 确保麦克风已连接
3. 尝试刷新页面重新授权

### 问题2：语音识别不准确
**解决方案**:
1. 说话更清晰
2. 减少背景噪音
3. 调整麦克风音量
4. 使用Chrome或Edge浏览器

### 问题3：考拉不说话
**解决方案**:
1. 检查浏览器是否支持语音合成
2. 检查系统音量设置
3. 查看浏览器控制台是否有错误

### 问题4：识别的文字不显示
**解决方案**:
1. 确保说话时间足够长（至少1-2秒）
2. 停止录音后等待几秒
3. 检查网络连接
4. 刷新页面重试

## 🌟 示例对话

### 对话1：问候
```
你: Hello!
考拉: Hello! How are you today? I'm excited to practice English with you!

你: I'm fine, thank you.
考拉: That's great! Can you tell me more about that?
```

你: I'm from Beijing.
考拉: Interesting! How do you feel about it?
```

考拉: Excellent! Can you tell me about your favorite book?
```

## 📱 移动设备使用

### 手机/平板
1. 使用Chrome或Safari浏览器
2. 允许麦克风权限
3. 竖屏使用效果更好
4. 确保网络连接稳定

### 注意事项
- 移动设备的语音识别可能略慢
- 建议使用耳机以获得更好的音质
- 避免在嘈杂环境中使用
