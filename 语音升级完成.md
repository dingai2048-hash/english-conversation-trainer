## 🚀 立即使用

### 3步配置

#### 1. 打开设置
点击右上角"设置"按钮

#### 2. 配置 OpenAI TTS
滚动到 "🔊 语音合成设置"：
- 语音服务商：选择 **"OpenAI TTS (推荐)"**
- OpenAI API Key：输入您的 Key（与对话AI共用）
- 音色选择：选择 **"Nova"**（友好活泼）
- 语音质量：选择 **"HD 高清质量"**
- 语速调节：设置 **0.9x**（稍慢，适合学习）

#### 3. 保存并测试
- 点击"保存设置"
- 刷新浏览器
- 开始对话，听考拉用自然的声音说话！
