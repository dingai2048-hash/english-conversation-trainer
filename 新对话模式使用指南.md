# 新对话模式使用指南

## 🎉 全新会话式对话体验

我们已经将应用升级为更自然的会话式对话模式！不再需要每次都点击麦克风，现在可以像真实对话一样流畅交流。

## 📱 如何使用

### 1. 开始对话

- 打开应用，你会看到一个**蓝色麦克风按钮**
- 按钮下方显示"点击开始新对话"
- 点击按钮后：
  - 按钮变为**红色停止按钮**
  - 自动开始录音
  - 你可以开始说话了！

### 2. 进行对话

- 说完一句话后，系统会自动识别并提交
- AI考拉会回复你
- 回复结束后，你可以继续说话
- 整个过程无需再次点击按钮

### 3. 结束对话

- 当你想结束这次对话时，点击**红色停止按钮**
- 系统会：
  - 自动生成对话摘要（用AI总结你们聊了什么）
  - 保存完整对话到历史记录
  - 返回初始状态

### 4. 查看历史

- 左侧（桌面端）或历史标签（移动端）显示所有历史对话
- 每条记录显示：
  - 📅 日期和时间
  - 📝 AI生成的摘要
  - 💬 消息数量
- **点击任意历史记录**可以查看完整对话内容

## 🆚 新旧模式对比

### 旧模式（点击说话）
```
点击麦克风 → 说话 → 自动提交 → AI回复 → 点击麦克风 → 说话 → ...
```
每次都要点击，比较繁琐

### 新模式（会话式）
```
点击开始 → 说话 → AI回复 → 说话 → AI回复 → ... → 点击结束
```
一次点击，持续对话，更自然！

## ✨ 主要优势

1. **更自然的对话体验**
   - 不需要频繁点击
   - 像真实对话一样流畅

2. **智能摘要**
   - AI自动总结每次对话
   - 方便快速回顾

3. **完整历史记录**
   - 可以查看任何历史对话的完整内容
   - 包括时间戳和翻译

4. **清晰的会话边界**
   - 每次对话都是独立的会话
   - 有明确的开始和结束

## 💡 使用技巧

### 连续对话模式

- 在设置中可以启用"连续对话模式"
- 启用后，AI回复完会自动开始录音
- 适合想要快速练习的用户

### 查看翻译

- 右上角可以切换"中文翻译"开关
- 开启后，所有AI回复都会显示中文翻译
- 历史记录中也会保存翻译

### 最佳实践

1. **每次对话聚焦一个主题**
   - 这样AI生成的摘要更准确
   - 方便以后查找

2. **适时结束对话**
   - 不要让单次对话太长
   - 建议5-10轮对话后结束

3. **定期查看历史**
   - 回顾之前的对话
   - 看看自己的进步

## 🔧 故障排除

### 问题：点击开始后没有反应

**解决方案**：
- 检查麦克风权限
- 确保使用Chrome或Edge浏览器
- 刷新页面重试

### 问题：对话没有保存到历史

**解决方案**：
- 确保点击了"结束对话"按钮
- 检查浏览器的localStorage是否被禁用
- 至少要有一条消息才会保存

### 问题：历史记录点击没反应

**解决方案**：
- 刷新页面重试
- 检查浏览器控制台是否有错误
- 确保历史记录确实存在
